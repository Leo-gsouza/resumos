Exercicio 01

Crie um programa que leia um número real qualquer pelo teclado e mostre na tela a sua porção Inteira.

Clique para ver a resolução

                    
HTML
label for="numeroId">Numero Real: 
input type="number" name="numero" id="numeroId" oninput="ex001(um)"
div class="resultado" id="um">

SCRIPT
function ex001(res){
    let numero = document.getElementById('numeroId').value
    res.innerHTML = `Valor inteiro é ${parseInt(numero)}`
    
}


Explicação

HTML
Foi inserido um oninput no input:number
com o nome da função ex001 com o parametro
um que é o Id da div para exibir o resultado.
O oninput dá o resultado instantaneo

SCRIPT
Foi inserido na variavel numero o valor 
do input:number por meio do Id.
Para inserir um valor inteiro foi usado 
o parseInt que exibe apenas numeros inteiros

Exercicio 2

Faça um programa que leia o comprimento do cateto oposto e do cateto adjacente de um triângulo retângulo. Calcule e mostre o comprimento da hipotenusa.

Clique para ver a resolução

                    
HTML
label for="opostoId">Oposto: 
input type="number" name="oposto" id="opostoId" oninput="ex002(opostoId, adjacenteId, dois)"
label for="adjacenteId">Adjacente:
input type="number" name="adjacente" id="adjacenteId" oninput="ex002(opostoId, adjacenteId, dois)"
div class="resultado" id="dois"  

SCRIPT
function ex002(opo, adj, res){
    let adjacente = Number(adj.value)
    let oposto = Number(opo.value)
    res.innerHTML = `Hipotenusa = ${((oposto**2 + adjacente**2) **0.5).toFixed(2)}`
}


Explicação

HTML
Foi usado dois input:number com oninput
Oninput com a função ex002 e tres parametros
sendo ele, os dois inputs e a div que exibe
o resultado.

JAVASCRIPT 
Criado duas variaveis com os valores
dos inputs. Foi realizado uma operação 
matematica para encontrar o valor e exibir
como resposta

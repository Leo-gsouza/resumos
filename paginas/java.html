    let radianos = angulo*3.141592653/180

Exercicio 04

Um professor quer sortear um dos seus quatro alunos para apagar o quadro. Faça um programa que ajude ele, lendo o nome dos alunos e escrevendo na tela o nome do escolhido.

Clique para ver a resolução

                    
HTML

    label for="aluno1Id">Aluno 1:
    input type="text" name="aluno1" id="aluno1Id">

    label for="aluno2Id">Aluno 2:

function ex004(resultado){
    const aluno1 = document.getElementById('aluno1Id').value
    const aluno2 = document.getElementById('aluno2Id').value
    const aluno3 = document.getElementById('aluno3Id').value
    const aluno4 = document.getElementById('aluno4Id').value

    const alunos = []
    if (aluno1) alunos.push(aluno1)
    if (aluno2) alunos.push(aluno2)
    if (aluno3) alunos.push(aluno3)
    if (aluno4) alunos.push(aluno4)

    if (alunos.length > 0){
        const indice = Math.floor(Math.random() * alunos.length)
        const sorteado = alunos[indice]
        resultado.innerHTML =  `${sorteado}`
    }
}   


Explicação

HTML 
1-Criado 4 input:texto para inserir
o nome dos quatro alunos.
2-Cada input tem um Id para ser chamado

3-Criado um input:submit que é um botão

4-Criado a div com o Id quatro para exibir

SCRIPT 
1-Criado 4 const com os Ids dos input:texto
que representam o nome dos quatro alunos
2- Criado uma lista vazia
3- Usado a 
    condição SE para inserir o aluno
    na lista
 caso ele exista, usando o push:
lista.push(variavel)
4-Usado a 
    condição SE para verificar se a
    lista é maior do que 0

5- Criado uma 
    const chamada de indice que
    tem como objetivo retornar um indice
 dentro 
da lista, se a lista tiver 5 elementos, vai

6-Criado uma const com o nome sorteado que 
carrega a lista alunos com o indice para

7- Por fim foi 
    usado o parametro resultado
    que representa a div
 para exibir o resultado 
com innerHTML 


Aprofundar :

const indice = Math.floor(Math.random() * alunos.length)

analisar ponto por ponto do codigo
para enteder melhor como ele é capaz
de sortear um valor dentro da lista

Math.floor() = arredonda o valor para cima
ou para baixo.

Math.Random() = seleciona um valor enteder
0 e 1 que pode ser 0.2563 ou 0.852 etc

alunos.lenght = o número de elementos dentro
da lista.

O que acontece com esse codigo:
o valor gerado por Math.random é
multiplicado pelo numero de elementos 
dentro da lista, exemplo: 
    0.256542 * 4
    que vai retornar 1.026164.


Em seguida o Math.floor vai arredondar.
Como o valor está mais proximo de 1, a 
cont indice vai valer 1

Exercicio 06

Crie um programa que leia o nome completo de uma pessoa e mostre:

  1. O nome com todas as letras maiúsculas e minúsculas.
  2. Quantas letras ao todo (sem considerar espaços).
  3. Quantas letras tem o primeiro nome.

Clique aqui para ver a resolução

                    

HTML

label for="nome" class="nome">Nome:
input type="text" name="nome" id="nomeCompleto" oninput="ex006(seis)" placeholder="Nome Completo">
div class="resultado" id="seis">

SCRIPT
function ex006(resultado){
    const nome = document.getElementById('nomeCompleto').value
    
    const primeiro = nome.slice(0, nome.indexOf(' '))
    const ultimo = nome.slice(nome.lastIndexOf(' ') + 1)
    const maiusculo = nome.toUpperCase()
    const minusculo = nome.toLowerCase()
    const espacos = nome.split(' ').length -1
    const caracteres = nome.length - espacos

    resultado.innerHTML = `Primeiro nome: ${primeiro}
    Ultimo nome: ${ultimo} 
    Maiusculo: ${maiusculo}
    Minusculo: ${minusculo} 
    Letras: ${caracteres}`
}

HTML

exibir o resultado simutaneamente

SCRIPT 
1- criado 7 constantes 
2 - Extrair o primeiro nome foi usado
slice seguido de 0 que indica o primeira
letra até o primeiro espaço. O ultimo nome é usado
o slice que inicia em um indice e vai
ate o fim se este não for informado e 
dentro do slice é usando o lastIndexOf(' ')
com um espaço dentro, o que indica que é
buscado o indice do ultimo espaço. O + 1
é para somar com o indice do espaço e iniciar 
na letra seguinte
3-Deixar o nome em maiusculo é usado 
a variavel seguida de .toUpperCase() e

4-Metodo split transforma a variavel

e subtrair 1, pois este sera o valor
de espaços entre a variavel. 
5- Para saber o numero de letras total 
foi ultizado o lenght na variavel nome

Exercicio 10

Faça um programa que leia uma frase pelo teclado e mostre quantas vezes aparece a letra “A”, em que posição ela aparece a primeira vez e em que posição ela aparece a última vez.

Clique aqui para ver a resolução

                    
HTML 
div class="js">
    label for="fraseId10">Frase: /label>
    input type="text" name="frase" id="fraseId10">
    input type="submit" value="Buscar" onclick="ex010(dez)">
    div class="resultado" id="dez">/div>
/div>

JAVASCRIPT 
function ex010(resultado){
    let frase = document.getElementById('fraseId10').value.toLowerCase()
    frase = frase.replace(/[âáãà]/g, 'a')
    frase = frase.replace(/ /g, '')
    let letraA = frase.split('a').length - 1
    let primeira = frase.indexOf('a') + 1
    let ultima = frase.lastIndexOf('a') + 1 
    resultado.style.display = 'block'

    resultado.innerHTML = `p>Quantidade de letras 'a' = ${letraA}/p>\
    p>1ª letra 'a' está na posição ${primeira}/p>\
    p>Ultima letra 'a' está na posição ${ultima}/p>` 
}


Explicação

HTML

2 - Input Submit com uma 
    função
    click ex010()
 com um parametro 
que é a Div para exibir o resultado
3 - Div para exibir o resultado

JAVASCRIPT 
1 - Variavel com Id do input texto 
2 - Replace para substituir todos 
o 'a' com acentos pelo 'a' comum.
variavel.replace(/[âáãà]/g, 'a')
3 - Replace para substituir todos 
os espaços por nada, ou seja, 
remover os espaços. 
variavel.replace(/ /g, '')
4 - Split('a') cria uma lista com 
removendo o caractere escolhido que 
neste caso é o a. 
Exemplo: a vida é boa. O split('a') 
vai criar uma lista ['vid', 'é', 'bo']
Quando o lenght é adicionado ele conta 
o numero de vezes que o caractere que 
está dentro de split ocorre, mas a 
contagem dá 1 a mais, por isso utiliza 
o - 1
5 - indexOf() procurar a primeira 
ocorrencia do caractere selecionado
como a psição se inicia do 0 é usado 
+ 1 para dar a posição comum que é 
iniciar do 1
6 - lastIndexOf('a') vai dar a ultima 
ocorrencia do caractere. o uso do 
+ 1 é semelhante o do ponto anterior

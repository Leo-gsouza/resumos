Exercicio 03

Faça um programa que leia um ângulo qualquer e mostre na tela o valor do seno, cosseno e tangente desse ângulo.

Solução

                    
from math import sin, radians, cos, tan

angulo = int(input('Digite o angulo que você deseja: '))
seno = sin(radians(angulo))
coseno = cos(radians(angulo))
tangente = tan(radians(angulo))

print(f'O seno do angulo de {angulo}º é {seno:.2f}.\
\nO coseno do angulo de {angulo}º é {coseno:.2f}.\
\nA tangente do angulo de {angulo}º é {tangente:.2f} ')


Explicações

Foi necessário importar da biblioteca math 
os métodos sin, cos, tan e radians.

Usou-se \ para continuar o código na linha 
seguinte e \n para exibir o texto em uma nova 
linha, evitando novos prints.

Exercicio 06

Crie um programa que leia o nome completo de uma pessoa e mostre:

  1. O nome com todas as letras maiúsculas e minúsculas.
  2. Quantas letras ao todo (sem considerar espaços).
  3. Quantas letras tem o primeiro nome.

Solução

                    
nome = str(input('Nome completo: ')).strip().title()
primeiro = nome[:nome.find(' ')]
ultimo = nome[nome.rfind(' '):]
maiusculo = nome.upper()
minusculo = nome.lower()
espacos = nome.count(' ')
letras = len(nome) - espacos

print(f'Primeiro nome: {primeiro}\
       \nUltimo nome: {ultimo}\
       \nMaiusculo: {maiusculo}\
       \nMinusculo: {minusculo}\
       \nQuatidade de letras: {letras}')


Explicação


Após a variável nome ser inserida, foi 
utilizado strip para remover espaços 
digitados no início e fim, e title para 
deixar todas as primeiras letras em maiúsculo.

A variável primeiro foi criada para obter 
o primeiro nome. Após o nome da variável, 
foi inserido um colchete seguido de vazio 
antes dos dois pontos, que significa desde 
o início até find(' '), que indica o 
primeiro espaço encontrado.

A variável ultimo foi criada semelhante à 
variável primeiro. A única diferença é que 
foi utilizado rfind, que procura da direita 
para a esquerda, encontrando assim o último 
espaço, e a ordem dentro dos colchetes, 
que se inicia do último espaço até o fim.

Para obter a quantidade de letras, foi 
usado count(' ') para contar o número de 
espaços e len(nome), que conta o número 
de caracteres da variável. Logo em seguida, 
foi realizada a subtração entre o número 
de caracteres e os espaços.

Para deixar o nome em maiúsculo e minúsculo, 
basta utilizar após o nome .upper() para 
maiúsculo e .lower() para minúsculo.

Exercicio 13

Escreva um programa que leia a velocidade de um carro. Se ele ultrapassar 80Km/h, mostre uma mensagem dizendo que ele foi multado. O valor da multa é de R$163,50 e R$15 por quilometro caso utrapasse 10% acima da velocidade limite

Solução

                    
LIMITE = 80
velocidade = int(input('Velocidade (Km/h): '))

if velocidade > LIMITE:
        print(f'Utrapassou o limite de velocidade!')
        
        if velocidade <= (LIMITE * 1.1):
            print(f'Valor da multa: R$163.50')

        elif velocidade > (LIMITE * 1.1):
            print(f'Valor da multa:\
            {163.50 + ((velocidade - (LIMITE * 1.1)) * 15)}')
    
else:
        print('Não utrapassou o limite de velocidade')


    
Explicação


1 - Constante limite
2 - Variavel para inserir a velocidade
3 - Condição SE a velocidade é maior 
do que o limite, vai exibir a mensagem 
que utrapassou o limite. 
4 - Condição SE dentro da outra condição. 
Se a velocidade for menor do que o limite
+ 10% a multa será de um valor fixo
5 - Se a velocidade for maior do o
     limite
    + 10%
 a multa será o valor fixo mais 15 
reais por km utrapassado
6 - Else para fecha a condição caso a 
velocidade seja inferior ao limite

Exercicio 16

Faça um programa que leia um ano qualquer e mostre se ele é bissexto.

Solução

                    
ano = int(input('Digite o ano: '))

if (ano % 4 == 0 and ano % 100 != 0) or (ano % 400 == 0):
    print(f'O ano de {ano} é bissexto')
else:
    print(f'O ano de {ano} não é bissexto')


Explicação

1 - Criado uma variavel ano
2 - Condição Se com AND e OR
para verificar a ocorrencia
3 - Se o resto da divisão por 4
for igual a 0 E o resto da divisão 
por 100 for diferente de 0
 é bissexto, OU se o resto da 
divisão por 400 for igual a 0 
também é bissexto.


Detalhes

A necessidade do uso de AND e OR 
foi por causa da regra do ano bissexto.
Para ser um ano bissexto precisa ser 
divisivel por 4 e não pode ser divisivel
por 100, exceto se for divisivel por 400.
Repare que qualquer valor divisivel por
400 vai ser divisivel por 100 também.

Exemplo: Ano 400
o ano 400 é divisivel por 4, mas também
é divisivel por 100, então a condição 
seria falsa pois precisa ser divivel por 4
E não divisivel por 100. Ai que entra o OU
perguntando: É divisivel por 400. Sim, entáo
é bissexto

    maior = n1
elif n2 > n1 and n2 > n3:
    maior = n2

Foi utilizado tres soluções
1ª Usando condição SE para 
verificar quem é o maior número.
mais complexa porque exige que 
crie o código.

Esta função procura o maior valor 
entre as variaveis que foram inseridas
dentro dela ()
3ªUsando uma lista. É semelhante a 
anterior usando MAX(lista), mas ao 
inves de colocar as variaveis dentro ()
basta apena sinserir a lista

Explicação



    1- Criado uma variavel salario

    é menor ou maior que
    3 - opcional - variavel convertida em
    STRING usando format e com somente
    duas casas decimais (:.2f)
    4 - opcional - REPLACE para trocar
    o ponto(.) por virgula(,)




Detalhe

"A conversão para string e o uso de 
replace são opcionais. Não é possível 
aplicar replace em um valor float, 
por isso a conversão para string foi 
necessária, junto com a formatação 
para duas casas decimais. Em seguida, 
o ponto foi trocado por vírgula, 
apenas para fins estéticos, exibindo 
o resultado com duas casas decimais

Exercicio 21

Escreva um programa em Python que leia um número inteiro qualquer e peça para o usuário escolher qual será a base de conversão: 1 para binário, 2 para octal e 3 para hexadecimal.

Solução

                    
numero = int(input('Digite um número: '))

conversao = int(input('Qual a base de conversão: \
                        \n[1]Binário \n[2]Octal\
                        \n[3]Hexadecimal \n>>'))

if conversao == 1:
    convertido = bin(numero)[2:]
    info = 'binário'
elif conversao == 2:
    convertido = oct(numero)[2:]
    info = 'octal'
elif conversao == 3:
    convertido = hex(numero)[2:]
    info = 'hexadecimal'

print(f'O número {numero} em {info} é {convertido}')


Explicação

1 - Criado duas variaveis
2 - Condição SE para 3 opções
3 - Usado a função bin(), oct() e hex() 
para converter o número
4 - [2:] Para iniciar após o segundo 
caractere

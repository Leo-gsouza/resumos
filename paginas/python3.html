Exercicio 01

1º Crie um programa que leia um número real qualquer pelo teclado e mostre na tela a sua porção Inteira.

Clique aqui para ver a resolução

                    
from math import trunc


numero = float(input('Digite um número real: '))

print(f'O numéro inteiro é {int(numero)}')

print(f'O número inteiro é {trunc(numero)} ')


Explicações


Este exercício foi resolvido de duas maneiras:

1ªUsando o método int, que mostra apenas 
a parte inteira de um número.

2ª Usando o método trunc da biblioteca math, 
que também pega apenas a parte inteira.

Exercicio 02

2º Faça um programa que leia o comprimento do cateto oposto e do cateto adjacente de um triângulo retângulo. Calcule e mostre o comprimento da hipotenusa.

Clique aqui para ver a resolução

                    
oposto = float(input('Cateto Oposto: '))
adjacente = float(input('Cateto Adjacente: '))

print(f'Hipotenusa = {(oposto**2 + adjacente**2)**0.5:.2f}')


Explicações

Este exercício simples usa duas variáveis. 
O valor da hipotenusa foi calculado diretamente, 
sem a necessidade de criar outra variável. 
Foi usado :.2f para formatar a saída com dois 
números após a vírgula.

Exercicio 03

3º Faça um programa que leia um ângulo qualquer e mostre na tela o valor do seno, cosseno e tangente desse ângulo.

Clique aqui para ver a resolução

                    
from math import sin, radians, cos, tan

angulo = int(input('Digite o angulo que você deseja: '))
seno = sin(radians(angulo))
coseno = cos(radians(angulo))
tangente = tan(radians(angulo))

print(f'O seno do angulo de {angulo}º é {seno:.2f}.\
\nO coseno do angulo de {angulo}º é {coseno:.2f}.\
\nA tangente do angulo de {angulo}º é {tangente:.2f} ')


Explicações

Foi necessário importar da biblioteca math 
os métodos sin, cos, tan e radians.

Usou-se \ para continuar o código na linha 
seguinte e \n para exibir o texto em uma nova 
linha, evitando novos prints.

Exercicio 04

4º Um professor quer sortear um dos seus quatro alunos para apagar o quadro. Faça um programa que ajude ele, lendo o nome dos alunos e escrevendo na tela o nome do escolhido.

Clique aqui para ver a resolução

                    
from random import choice

alunos = ['Leo','Carlos','Nat', 'Ana' ]
escolhido = choice(alunos)

print(f'Escolhido = {escolhido}')


Explicações

Usado o metodo CHOICE da biblioteca RANDOM
Criado uma lista com 4 elementos
Criado uma variavel para a escolha

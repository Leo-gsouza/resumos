Exercicio 01

Crie um programa que leia um número real qualquer pelo teclado e mostre na tela a sua porção Inteira.

Clique aqui para ver a resolução

                    
from math import trunc


numero = float(input('Digite um número real: '))

print(f'O numéro inteiro é {int(numero)}')

print(f'O número inteiro é {trunc(numero)} ')


Explicações


Este exercício foi resolvido de duas maneiras:

1ªUsando o método int, que mostra apenas 
a parte inteira de um número.

2ª Usando o método trunc da biblioteca math, 
que também pega apenas a parte inteira.

Exercicio 03

Faça um programa que leia um ângulo qualquer e mostre na tela o valor do seno, cosseno e tangente desse ângulo.

Clique aqui para ver a resolução

                    
from math import sin, radians, cos, tan

angulo = int(input('Digite o angulo que você deseja: '))
seno = sin(radians(angulo))
coseno = cos(radians(angulo))
tangente = tan(radians(angulo))

print(f'O seno do angulo de {angulo}º é {seno:.2f}.\
\nO coseno do angulo de {angulo}º é {coseno:.2f}.\
\nA tangente do angulo de {angulo}º é {tangente:.2f} ')


Explicações

Foi necessário importar da biblioteca math 
os métodos sin, cos, tan e radians.

Usou-se \ para continuar o código na linha 
seguinte e \n para exibir o texto em uma nova 
linha, evitando novos prints.

Exercicio 05

O mesmo professor do desafio 19 quer sortear a ordem de apresentação de trabalhos dos alunos. Faça um programa que leia o nome dos quatro alunos e mostre a ordem sorteada.

Clique aqui para ver a resolução

                    
from random import shuffle

alunos = [str(input('1º Aluno: ')), 
          str(input('2º Aluno: ')), 
          str(input('3º Aluno: ')),
          str(input('4º Aluno: '))]

shuffle(alunos)

c = 1

for a in alunos:
    print(f'{c}º Aluno é {a}')
    c += 1


Explicação

Foi necessario importar o metodo shuffle
da biblioteca random. Ele muda a ordem
dos elementos dentro da lista.

O input criado para iserir os valores em 
alunos foram criados dentro de uma lista
para reduzir o codigo.

criado uma variavel c para indicar o 
número do aluno dentro do for.

Usado o for para interar a lista. O valor 
'a' sempre exibira todos os elementos 
dentro da lista um por vez.

Exercicio 06

22: Crie um programa que leia o nome completo de uma pessoa e mostre:

  1. O nome com todas as letras maiúsculas e minúsculas.
  2. Quantas letras ao todo (sem considerar espaços).
  3. Quantas letras tem o primeiro nome.

Clique aqui para ver a resolução

                    
nome = str(input('Nome completo: ')).strip().title()
primeiro = nome[:nome.find(' ')]
ultimo = nome[nome.rfind(' '):]
maiusculo = nome.upper()
minusculo = nome.lower()
espacos = nome.count(' ')
letras = len(nome) - espacos

print(f'Primeiro nome: {primeiro}\
       \nUltimo nome: {ultimo}\
       \nMaiusculo: {maiusculo}\
       \nMinusculo: {minusculo}\
       \nQuatidade de letras: {letras}')


Explicação


Após a variável nome ser inserida, foi 
utilizado strip para remover espaços 
digitados no início e fim, e title para 
deixar todas as primeiras letras em maiúsculo.

A variável primeiro foi criada para obter 
o primeiro nome. Após o nome da variável, 
foi inserido um colchete seguido de vazio 
antes dos dois pontos, que significa desde 
o início até find(' '), que indica o 
primeiro espaço encontrado.

A variável ultimo foi criada semelhante à 
variável primeiro. A única diferença é que 
foi utilizado rfind, que procura da direita 
para a esquerda, encontrando assim o último 
espaço, e a ordem dentro dos colchetes, 
que se inicia do último espaço até o fim.

Para obter a quantidade de letras, foi 
usado count(' ') para contar o número de 
espaços e len(nome), que conta o número 
de caracteres da variável. Logo em seguida, 
foi realizada a subtração entre o número 
de caracteres e os espaços.

Para deixar o nome em maiúsculo e minúsculo, 
basta utilizar após o nome .upper() para 
maiúsculo e .lower() para minúsculo.
